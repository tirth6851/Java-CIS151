# Java CIS151 - Introduction to Computing

## Overview
This repository contains coursework and projects from CIS 151 (Introduction to Computing with Java) at Cleveland State University. The repository includes implementations of fundamental programming concepts including loops, arrays, methods, and object-oriented programming principles.

## Course Topics
- **Basic Java Syntax**: Variables, data types, operators, and control structures
- **Loops**: For loops, while loops, nested loops, and iteration patterns
- **Arrays**: Single and multi-dimensional arrays, array manipulation
- **Methods**: Function design, parameters, return values, and method overloading
- **Object-Oriented Programming**: Classes, objects, encapsulation, and inheritance basics

## Repository Structure
- `labs/` - Laboratory exercises and assignments
- `brocode/` - Practice exercises and examples
- `tempT/` - Temporary testing files
- `target/` - Compiled output files

## Technologies
- **Language**: Java
- **Build Tool**: Maven (pom.xml included)
- **Version Control**: Git and GitHub

## Learning Objectives
This coursework demonstrates progressive learning in:
- Problem-solving with code
- Algorithmic thinking
- Software development best practices
- Version control workflows

## About
Created as part of the Computer Science curriculum at Cleveland State University. This repository serves as a portfolio of fundamental Java programming skills and computer science concepts.
