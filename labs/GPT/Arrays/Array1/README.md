# Array Lab 1

## Objective
Learn how arrays work in Java.

## Tasks
1. Create an integer array with 5 numbers
2. Print the first element
3. Print the last element
4. Print all elements using a loop

## Rules
- Array index starts at 0
- File name must match class name
- Do not hardcode indexes inside the loop

## What you should understand after this lab
- How arrays store data
- How indexing works
- How loops and arrays connect
