# Array Lab 2

## Objective
Practice creating and using arrays with loops in Java.

## Tasks
1. Create an integer array of size 5
2. Fill the array using a loop with values:
   10, 20, 30, 40, 50
3. Calculate the sum of all elements
4. Print the sum

## Rules
- Use loops (no hardcoding values one by one)
- File name must match class name
- Code must compile and run without errors

## Expected Output
150
