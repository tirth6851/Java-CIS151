# Performance Report

## Objective
Analyze a set of performance scores and generate a structured summary report using arrays, loops, and conditional logic.

## Description
This program processes a fixed set of numeric performance scores. It calculates overall metrics and classifies each score into defined performance categories.

## Tasks
- Store performance scores in an array
- Calculate the total and average score
- Classify scores into performance categories:
  - Excellent (≥ 80)
  - Satisfactory (50–79)
  - Needs Improvement (< 50)
- Count how many scores fall into each category
- Print a formatted summary report

## Rules
- Use loops and conditional statements
- Do not hardcode totals or counts
- Use a single array
- File name must match the class name

## Expected Output
Total: 449  
Average: 64  
Excellent: 2  
Satisfactory: 3  
Needs Improvement: 2
